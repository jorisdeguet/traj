Bienvenue dans Traj

Traj est une interface web pour définir votre trajectoire sur la planète, dans le temps.

Traj ne stocke aucune information en dehors de votre navigateur. Bas besoin de compte ou de mot de passe. Nous utilisons soit le stockage du navigateur soit un fichier que vous sauvez où vous le souhaitez.

Vous pouvez partager une partie de votre trajectoire par courriel avec vos proches.
